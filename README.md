# Pratham-Dada-Portfolio
# 🚀 Personal Portfolio Website

Welcome to the repository for my personal portfolio website! This project serves as a digital hub to showcase my web development journey, featured projects, skills, and creations. It is designed to be clean, responsive, and highly interactive.
✨ Live Demo: [https://Pratham-Dada-Portfolio.github.io]

---

## 🛠️ Features

- Responsive Design: Fully optimized for mobile, tablet, and desktop viewports.
- Project Showcase: A clean grid layout displaying my latest web development applications.
- Interactive UI: Smooth scrolling, dynamic elements, and a clean, modern aesthetic.
- Contact Setup: An easy way for visitors, peers, and collaborators to get in touch.

---

## 💻 Tech Stack

This website is built entirely from scratch using core web technologies:

- HTML5: Structuring the content and semantic layout.
- CSS3: Custom styling, layouts (Flexbox/Grid), and animations.
- JavaScript (ES6): Dynamic behavior and interactive UI elements.
